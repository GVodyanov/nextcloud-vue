This repo contains the various Vue.js components that Nextcloud uses for its internal design and structure.

# Installation

```sh
npm i --save nextcloud-vue
```

# Usage
```js
import { AppNavigation } from 'nextcloud-vue'
```

# List of the components
- AppNavigation
- DatePicker
- PopoverMenu
- Multiselect
- Avatar
- and many more...
